### Housing

Objective: 캘리포니아 인구조사 데이터를 사용해 캘리포니아의 주택 가격 모델을 만드는 것.
다른 측정 데이터가 주어졌을 때 구역의 중간 주택 가격을 예측해야 한다.

지도학습, 회쉬문제, 데이터의 크기가 그렇게 크지 않으므로 일반적인 Batch 방법을 사용하여 학습.
